* RF1 - O sistema deve permitir o cadastro dos hóspedes com informações como nome, CPF e telefone.  
* RF2 - O sistema deve validar a numeração do CPF.  
* RF3 - O sistema deve ter registrado os produtos do frigobar.  
* RF4 - O sistema deve impedir duas reservas do mesmo quarto.  
* RF5 - O sistema deve informar o status de cada quarto, ocupado, livre ou sujo.  
* RF6 - O sistema deve atualizar automaticamente check-in e check-out dos quartos.  
* RF7 - O sistema deve fazer automaticamente o calculo dos valores a serem cobrados.  
* RF8 - O sistema deve registrar todo o faturamento e despesas do hotel.  
* RF9 - O sistema deve permitir identificar facilmente os diferentes tipos de quartos.  
* RF10 - O sistema deve permitir a alteração nos dias reservados pelo hóspede.
